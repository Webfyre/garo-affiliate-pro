A powerful, scalable affiliate marketing plugin with tiered commissions, MLM bonuses, fraud detection, and an elegant dashboard UI.

== Description ==

Garo Affiliate Pro is a lightweight, scalable affiliate system built for high-converting shops.

**Features:**
* Easy affiliate signup and login
* Personalized dashboard (mobile-ready)
* 2-level MLM commissions (recruit bonus)
* Tiered payouts for super affiliates
* Click + conversion tracking
* Email notifications (signup, sales, payouts)
* Manual + auto payouts
* Admin reports and fraud detection
* Shortcodes and API integration

== Installation ==

== Frequently Asked Questions ==

= Can I track referrals for guest customers? =  
Yes, using browser cookies, device/IP detection, and fraud-prevention logic.

= Does this support subdomain installs like affiliates.garogiftshop.co.ke? =  
Yes, the plugin supports cross-domain tracking if configured properly.

= Is this compatible with WooCommerce and Blocksy? =  
Yes, it's tested with WooCommerce, Blocksy, and LiteSpeed Cache.
